# Tech Stack

## Backend
- **Framework**: Laravel 12.0
- **Language**: PHP 8.4
- **Livewire**: Version 4.1 (for standard Laravel server-side and some auth pages)
- **Wayfinder**: v0.1.16 for strongly typed routing
- **Fortify**: v1.30 as headless authentication backend
- **Database**: SQLite (default, typical in scaffolding), MySQL/PostgreSQL assumed in production

## Frontend
- **Framework**: React 19.2
- **Bridge**: Inertia.js (v3) + @inertiajs/react (v3.0.3)
- **Styling**: TailwindCSS v4.1 + DaisyUI v5.5
- **Tooling**: Vite 7.0 + TypeScript 6.0
- **Real-time**: Laravel Echo (v2.3) + Pusher JS

## Developer Tools
- **Static Analysis/Formatter**: Laravel Pint 1.24
- **Testing**: Pest PHP 4.3 + PHPUnit
- **Local Env**: Laravel Sail, Laravel Pail, Concurrent dev scripts
