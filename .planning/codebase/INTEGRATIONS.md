# Integrations

## Core Laravel Packages
- **Fortify**: Headless backend for auth logic (Registration, Login, Reset Password, Verification).
- **Inertia Laravel**: Server-side bridge for Inertia.
- **Wayfinder**: Generates typed TypeScript functions for backend routes, providing type-safe frontend routing without hardcoding URLs.

## Third-party / External
- **Pusher**: Configured for WebSocket based real-time events (`pusher/pusher-php-server`, `pusher-js`).
- **Spatie Translatable**: `spatie/laravel-translatable` for multilingual model properties.
- **Google Translate PHP**: `stichoza/google-translate-php`.
- **Mary UI**: `robsontenorio/mary` (v2.6) used potentially for Livewire-based UI components (e.g. admin panels).
- **DaisyUI**: TailwindCSS plugin for pre-built components on the React/Inertia side.
